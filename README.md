# flask框架搭建個人Blog

## 安裝nodejs npm
```shell
sudo apt install nodejs
sudo apt install npm
```
## 安裝mongodb on debian
ref :
https://docs.mongodb.com/manual/tutorial/install-mongodb-on-debian/

## 安裝PM2
```shell
npm install pm2 -g
pm2 completion install
```

## 安裝nginx
```shell
sudo apt install nginx
```

## gcp setting 
查看網路資料>防火牆>建立防火牆規則>
設定：方向（輸入）範圍：(0.0.0.0/0)
通訊協定和通訊埠：（tpc: xxxx）
